# Скрипты проекта

Эта папка содержит все вспомогательные shell-скрипты проекта.

## Основные скрипты

### Локальная разработка
- `start_local.sh` - Запуск проекта локально

### Деплой
- `deploy.sh` - Основной скрипт деплоя на сервер
- `deploy_to_server.sh` - Альтернативный скрипт деплоя
- `sync_to_server.sh` - Синхронизация файлов на сервер

### Проверка и диагностика
- `check_status.sh` - Проверка статуса сервисов
- `check_backend_status.sh` - Проверка статуса бэкенда
- `check_full_system.sh` - Полная проверка системы

### Резервное копирование
- `backup.sh` - Резервное копирование базы данных
- `backup_db.sh` - Альтернативный скрипт бэкапа

### Исправление проблем
- `fix_postgres_password.sh` - Исправление пароля PostgreSQL
- `fix_db_connection.sh` - Исправление подключения к БД

## Использование

Большинство скриптов можно запускать из корня проекта:

```bash
./scripts/start_local.sh
./scripts/deploy.sh
```

Или из папки scripts:

```bash
cd scripts
./start_local.sh
```
