# Шаблон наклеек для Word

## Как создать шаблон

1. Создайте документ Word с размером страницы A4
2. Создайте таблицу 2x4 (2 ряда, 4 колонки) для размещения 8 наклеек
3. Размер каждой ячейки: 52.5 мм × 148.5 мм
4. В каждой ячейке используйте следующие плейсхолдеры:

## Доступные плейсхолдеры

- `{{ company_name_ru }}` - Название компании на русском (ООО "Алмазгеобур")
- `{{ company_name_en }}` - Название компании на английском (LLP "Almazgeobur")
- `{{ nomenclature_name }}` - Название номенклатуры
- `{{ article }}` - Артикул
- `{{ matrix }}` - Типоразмер
- `{{ height }}` - Высота матрицы
- `{{ waterways }}` - Промывочные отверстия
- `{{ serial_number }}` - Серийный номер паспорта
- `{{ production_date }}` - Дата производства (год)
- `{{ order_number }}` - Номер заказа

## Для логотипа

Для добавления логотипа в шаблон:
1. Вставьте изображение-заглушку в нужное место
2. В коде логотип будет автоматически заменен на реальный

## Пример шаблона

```
┌─────────────────────────┐
│ {{ logo }}              │
│                         │
│ {{ nomenclature_name }} │
│                         │
│ Артикул: {{ article }}  │
│ Типоразмер: {{ matrix }}│
│ Высота: {{ height }} мм │
│                         │
│ Серийный номер:         │
│ {{ serial_number }}     │
│                         │
│ Дата: {{ production_date }}│
└─────────────────────────┘
```

## Размещение файла

Поместите файл шаблона по одному из путей:
- `backend/utils/templates/sticker_template.docx`
- `templates/sticker_template.docx`
- `/root/agb_passports/templates/sticker_template.docx`

## Примечания

- Шаблон должен быть в формате .docx
- Используйте синтаксис Jinja2 для плейсхолдеров
- Если шаблон не найден, будет использован стандартный метод генерации
