# Краткая инструкция: Создание шаблона наклеек

## Быстрый старт

1. **Создайте документ Word** с одной ячейкой таблицы размером **52.5 × 148.5 мм**

2. **Добавьте плейсхолдеры** в формате `{{ переменная }}`:
   - `{{ company_name_ru }}` - Название компании (рус)
   - `{{ company_name_en }}` - Название компании (англ)
   - `{{ nomenclature_name }}` - Название номенклатуры
   - `{{ article }}` - Артикул
   - `{{ matrix }}` - Типоразмер
   - `{{ height }}` - Высота матрицы
   - `{{ waterways }}` - Промывочные отверстия
   - `{{ serial_number }}` - Серийный номер
   - `{{ production_date }}` - Дата производства
   - `{{ order_number }}` - Номер заказа

3. **Сохраните как** `sticker_template.docx`

4. **Поместите файл** в:
   - `backend/utils/templates/sticker_template.docx` (локально)
   - `/root/agb_passports/backend/utils/templates/sticker_template.docx` (на сервере)

5. **Готово!** Система автоматически начнет использовать ваш шаблон при экспорте наклеек.

## Пример содержимого ячейки:

```
{{ company_name_ru }}
{{ company_name_en }}

{{ nomenclature_name }}

Артикул: {{ article }}
Типоразмер: {{ matrix }}
Высота: {{ height }} мм
Промывочные: {{ waterways }} мм

Серийный номер:
{{ serial_number }}

Дата: {{ production_date }}
```

## Важно

- Размер ячейки: **52.5 × 148.5 мм** (обязательно!)
- Отступы: **0 мм** везде
- Формат плейсхолдеров: `{{ переменная }}` (с двойными фигурными скобками)

Подробная инструкция: см. `ИНСТРУКЦИЯ_ШАБЛОН_НАКЛЕЕК.md`
